# Array

## Định nghĩa:
+ Tập hợp các phần tử cùng kiểu dữ liệu

## Phân loại
+ Mảng một chiều
    - example: arr[1] = 5
+ Mảng nhiều chiều
    - example: arr[row_idx][col_idx]

+ Khai báo:
    ```
    int[] intArr = {1, 2, 3, 4};
    or 
    int[] intArr = new int[4];
    ```
  
## Thao tác 

+ Duyệt mảng

+ Thêm/Xóa phần tử
